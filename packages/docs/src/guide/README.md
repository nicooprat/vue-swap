# Getting Started

## Installation

Install the `vue-swap` package:

```sh
npm i vue-swap
```

Or

```sh
yarn add vue-swap
```

## Quick start

Import the `Swap` component from `vue-swap`:

```vue{2,6}
<script>
import { Swap } from 'vue-swap'

export default {
  components: {
    Swap,
  },
}
</script>
```
